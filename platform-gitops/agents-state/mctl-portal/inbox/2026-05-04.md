## Top-3 (for spec-writer)

### 1. scaffolder-symlink-cve: Backstage Scaffolder symlink path traversal (CVE-2026-24046)
**Category:** security
**Impact:** 4
**Effort:** 1
**Rationale:** mctl-portal actively uses the scaffolder plugin to onboard services; this CVE allows any user with template creation or execution access to read, delete, or overwrite arbitrary files outside the sandbox workspace via crafted symlinks. The fix is a targeted patch-version bump of three packages (`@backstage/plugin-scaffolder-backend`, `@backstage/backend-defaults`, `@backstage/plugin-scaffolder-node`) with no breaking API changes. Leaving it unpatched exposes the backend filesystem and mctl-gitops commit credentials.
**Source:** https://github.com/backstage/backstage/security/advisories/GHSA-rq6q-wr2q-7pgp

### 2. nodejs-security-march-2026: Node.js 22 March 2026 security releases (CVE-2026-21637, CVE-2026-21710)
**Category:** security
**Impact:** 4
**Effort:** 2
**Rationale:** mctl-portal runs on Node.js 22 in production; two High-severity CVEs patch a remote TLS DoS and an `__proto__` header DoS that are reachable from the public-facing backend. Node.js 22.22.2 (LTS) ships all nine fixes and is a drop-in runtime upgrade requiring only the base image pin and a CI pipeline re-run. Five additional Medium CVEs (HTTP/2 memory leak, HashDoS, HMAC timing) are also resolved.
**Source:** https://nodejs.org/en/blog/vulnerability/march-2026-security-releases

### 3. backstage-v1-50-4-security-patch: Backstage v1.50.4 catalog security patch
**Category:** security
**Impact:** 3
**Effort:** 2
**Rationale:** This patch-level release directly addresses CVEs in the catalog-unprocessed-entities package family used by the portal's service catalog, the core feature of mctl-portal. As a patch (not major) release it does not trigger the ADR wait period, and community-plugins compatibility for patch releases is typically immediate. Upgrading also pulls in the v1.50.3 facets-endpoint performance improvement as a bonus.
**Source:** https://github.com/backstage/backstage/releases/tag/v1.50.4
