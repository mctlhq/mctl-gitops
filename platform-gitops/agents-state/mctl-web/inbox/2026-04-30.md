## Source: cve

- **CVE-2026-34404 (nuxt-og-image)** — Denial of Service via unbounded image dimensions in nuxt-og-image < 6.2.5; no restriction on width/height parameters allows resource exhaustion. Disclosed ~2026-03-31; fixed in nuxt-og-image 6.2.5. nuxt-og-image is a separate optional Nuxt module, not core nuxt. https://advisories.gitlab.com/pkg/npm/nuxt-og-image/CVE-2026-34404/

- **CVE-2026-34405 (nuxt-og-image)** — Reflected XSS via query parameter injection into HTML attributes in nuxt-og-image 6.1.2; improper neutralization of GET parameters enables client-side JavaScript execution by external users. Disclosed ~2026-03-31; fixed in nuxt-og-image 6.2.5. nuxt-og-image is a separate optional Nuxt module, not core nuxt. https://advisories.gitlab.com/pkg/npm/nuxt-og-image/CVE-2026-34405/

- **CVE-2026-0933 / GHSA-36p8-mvp6-cv38 (wrangler)** — OS Command Injection (CWE-78) in `wrangler pages deploy`; the --commit-hash parameter is passed unsanitized to a shell command, enabling arbitrary command execution in CI/CD pipelines where --commit-hash is populated from external sources. CVSS 3.1 critical (AV:N/AC:L/PR:L/UI:N/S:C/C:H/I:H/A:H); affects wrangler >= 2.0.15 < 3.114.17 and >= 4.0.0 < 4.59.1; disclosed 2026-01-21; fixed in 4.59.1. Current latest wrangler@4.87.0 is above the patched threshold. https://github.com/advisories/GHSA-36p8-mvp6-cv38

- **GHSA-f8mp-x433-5wpf / CVE-2023-7080 (wrangler)** — Historical High (CVSS 8.5): arbitrary remote code execution within `wrangler dev` Workers sandbox via V8 inspector listening on all network interfaces without Origin/Host header validation; affects wrangler >= 2.0.0 < 2.20.2 and >= 3.0.0 < 3.19.0; disclosed 2023-12-29. **Impact:** 5
**Effort:** 2
**Rationale:** CVE-2026-0933 (CVSS Critical) allows OS command injection via `wrangler pages deploy --commit-hash` which is populated from git in mctl-web's deploy.yml CI pipeline.
