## Top-3 (for spec-writer)

### 1. vue-patch-3-5-34: Vue 3.5.34 Patch Upgrade
**Category:** security  
**Impact:** 3  
**Effort:** 1  
**Rationale:** Vue 3.5.34 ships a DOM leak prevention fix, making this a security-adjacent patch. mctl-web is pinned at 3.5.30 — four patch releases behind — and running stale Vue core in a Nuxt SSG app means the DOM leak can surface on client-side hydration. Patching is semver-safe (patch bump) with no architectural impact. Existing proposal `vue-patch-3-5-33` targeted 3.5.33; 3.5.34 supersedes it and has not been proposed yet. Cross-checked ADRs: nothing in `context/decisions/` rejects Vue patch upgrades.  
**Source:** https://github.com/vuejs/core/releases/tag/v3.5.34

### 2. nuxt-upgrade-4-4-5: Nuxt 4.4.5 Patch Upgrade
**Category:** performance  
**Impact:** 2  
**Effort:** 2  
**Rationale:** Nuxt 4.4.5 (2026-05-10) introduces a caching layer for roots and short-circuits `isIgnored` relative lookups during SSG prerender, directly reducing build time and server cold-start latency for mctl-web's three prerendered routes. mctl-web is at 4.3.1 — more than a minor release behind. The most recent Nuxt upgrade proposal (`nuxt-upgrade-4-4-4`) targets 4.4.4; 4.4.5 is the current latest. Nuxt runs in the `admins` tenant with no memory pressure on `labs`. ADR 0001 explicitly requires tracking Nuxt ecosystem releases.  
**Source:** https://github.com/nuxt/nuxt/releases/tag/v4.4.5

### 3. workerd-runtime-2026-05-15: Workerd Runtime Update to v1.20260515.1
**Category:** performance  
**Impact:** 2  
**Effort:** 1  
**Rationale:** workerd v1.20260515.1 (2026-05-15) is the latest Cloudflare Worker runtime, 9 days newer than v1.20260506.1 covered by existing proposal `workerd-v8-148-upgrade`. Keeping the runtime pin current eliminates environment drift between local `wrangler dev` and production Cloudflare infrastructure and delivers any incremental V8 fixes shipped in the intervening dated releases. No Kubernetes resource impact; Worker runs entirely on Cloudflare. The workers-sdk `miniflare@4.20260515.0` confirms this version is already available.  
**Source:** https://github.com/cloudflare/workerd/releases/tag/v1.20260515.1
