## Top-3 (for spec-writer)

### 1. wrangler-full-upgrade: Upgrade wrangler from CVE floor to v4.86.0
**Category:** performance
**Impact:** 4
**Effort:** 1
**Rationale:** The existing `wrangler-cve-0933` proposal establishes a minimum of v4.59.1 to fix the critical OS command injection, but production must also reach v4.86.0 to eliminate resource leaks in worker teardown (stability risk in the `/api/*` worker) and gain stack traces in `wrangler tail` for faster incident diagnosis. The gap between 4.59.1 and 4.86.0 is 27 releases and constitutes meaningful accumulated technical debt on a core deploy tool.
**Source:** https://github.com/cloudflare/workers-sdk/releases/latest — wrangler@4.86.0, released 2026-04-28.

### 2. sass-deprecation-fix: Resolve sass v1.99.0 deprecation warnings before build failures
**Category:** refactor
**Impact:** 3
**Effort:** 2
**Rationale:** sass v1.99.0 (released 2026-04-02) deprecates several function naming patterns that mctl-web's stylesheets may rely on; deprecation warnings in sass are promoted to errors in subsequent major releases, which would break the `nuxt build` pipeline. Addressing them now, while they are still warnings, costs far less than an emergency fix during a future build outage. mctl-web uses sass 1.98.0 directly, so this is firmly in scope.
**Source:** https://github.com/sass/dart-sass/releases/latest — dart-sass v1.99.0, released 2026-04-02.

### 3. workerd-clearweak-migration: Migrate Cloudflare Worker off deprecated ClearWeak API
**Category:** refactor
**Impact:** 2
**Effort:** 2
**Rationale:** workerd v1.20260426.1 deprecates the ClearWeak V8 API in favor of its updated replacement; continued use of the deprecated path will produce runtime warnings and will eventually be removed in a future workerd release, breaking the Worker that handles all `/api/*` traffic including OAuth and tenant provisioning. Migrating now while the old API is still functional avoids a forced, potentially urgent migration later.
**Source:** https://github.com/cloudflare/workerd/releases/latest — workerd v1.20260426.1, released 2026-04-26.
