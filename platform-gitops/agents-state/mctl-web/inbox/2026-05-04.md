## Top-3 (for spec-writer)

### 1. wrangler-cve-2026-0933: Patch wrangler OS Command Injection (CVE-2026-0933)
**Category:** security
**Impact:** 4
**Effort:** 1
**Rationale:** Wrangler is used directly in our `deploy.yml` GitHub Actions workflow to deploy the Cloudflare Worker; CVE-2026-0933 (CVSS 7.7 High) enables OS Command Injection via an unsanitized `--commit-hash` argument in `wrangler pages deploy`. Upgrading to 4.87.0 (latest, well past the patched floor of 4.59.1) eliminates the vulnerability with a single version pin change and also brings in workerd v1.20260504.1 as a transitive update.
**Source:** https://github.com/cloudflare/workers-sdk/security/advisories/GHSA-36p8-mvp6-cv38

### 2. nuxt-minor-upgrade: Upgrade Nuxt from 4.3.1 to 4.4.4
**Category:** performance
**Impact:** 3
**Effort:** 2
**Rationale:** Nuxt 4.4.4 is a minor version bump over pinned 4.3.1 and is expected to include bug fixes and performance improvements without breaking changes; staying current reduces the risk of compounding upgrade debt as the Nuxt 4 ecosystem matures. The upgrade is low-risk given that our usage is constrained to SSG prerender for three routes plus the standard composable API, and no architectural constraints in ADR-0001 prohibit minor framework upgrades.
**Source:** https://github.com/nuxt/nuxt/releases/latest

### 3. patch-dep-refresh: Patch-level dependency refresh (vue-core, vueuse, sass)
**Category:** refactor
**Impact:** 2
**Effort:** 1
**Rationale:** Three in-stack dependencies have patch-level releases ahead of their pinned versions: Vue 3.5.33 (vs 3.5.30), @vueuse/core 14.3.0 (vs 14.2.1), and sass 1.99.0 (vs 1.98.0). Batching these into a single PR keeps the dependency graph current, reduces future upgrade friction, and costs near-zero effort since all are non-breaking patch or minor bumps. None of these changes affect Worker memory footprint and carry no risk for the `labs` tenant.
**Source:** https://github.com/vuejs/core/releases/latest; https://github.com/vueuse/vueuse/releases/latest; https://github.com/sass/dart-sass/releases/latest
