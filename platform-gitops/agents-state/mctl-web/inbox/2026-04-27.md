## Top-3 (for spec-writer)

### 1. wrangler-cve-0933: Upgrade wrangler to v4.59.1+ (patch CVE-2026-0933)
**Category:** security
**Impact:** 5
**Effort:** 1
**Rationale:** CVE-2026-0933 is OS command injection with CVSS 9.9 Critical in `wrangler pages deploy` via the `--commit-hash` parameter. mctl-web uses wrangler to deploy in GitHub Actions (`deploy.yml`), which makes the vulnerability a direct attack vector against the CI/CD pipeline. The fix is trivial: bump the wrangler version to v4.59.1 or above.
**Source:** https://github.com/advisories/GHSA-36p8-mvp6-cv38

### 2. nuxt-upgrade-4-4-2: Upgrade Nuxt to 4.4.2 and migrate to vue-router v5
**Category:** refactor
**Impact:** 4
**Effort:** 3
**Rationale:** vue-router v4.6.4 is the final release of the v4 branch (EOL); Nuxt 4.4.2 requires vue-router v5, and deferring the migration accumulates technical debt against a critical routing dependency. The migration also delivers up to 28x routing speed-up (unrouting) and typed layout props. The major vue-router upgrade requires verifying route configuration and composables, but the mctl-web codebase is small (3 pages).
**Source:** https://github.com/nuxt/nuxt/releases/latest; https://github.com/vuejs/router/releases/latest

### 3. vue-patch-3-5-33: Upgrade Vue to 3.5.33
**Category:** performance
**Impact:** 2
**Effort:** 1
**Rationale:** A Vue 3.5.33 patch release without breaking changes closes accumulated fixes since 3.5.30. The upgrade is risk-free, takes minutes, and is mandatory dependency hygiene before a larger Nuxt/vue-router upgrade.
**Source:** https://github.com/vuejs/core/releases/latest
