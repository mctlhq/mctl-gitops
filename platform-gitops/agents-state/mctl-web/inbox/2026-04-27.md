## Источник: cve

- **CVE-2026-0933 (wrangler)** — OS command injection в `wrangler pages deploy`: параметр `--commit-hash` передаётся напрямую в shell без санитизации, CVSS 3.1 = 9.9 Critical. Затронуты v2.0.15–3.114.16 и v4.0.0–4.59.0; fix в v3.114.17 и v4.59.1. https://github.com/advisories/GHSA-36p8-mvp6-cv38

- **CVE-2026-41248 (@clerk/nuxt)** — interpretation conflict в `createRouteMatcher` позволяет обойти middleware-защиту маршрутов без аутентификации, CVSS 9.1 Critical. Затронут `@clerk/nuxt` < 1.13.28 и < 2.2.2; опубликовано 15 апреля 2026. https://clerk.com/blog/middleware-based-route-protection-bypass

## Top-3 (для spec-writer)

### 1. wrangler-cve-0933: Обновление wrangler до v4.59.1+ (патч CVE-2026-0933)
**Категория:** security
**Impact:** 5
**Effort:** 1
**Обоснование:** CVE-2026-0933 — OS command injection с CVSS 9.9 Critical в `wrangler pages deploy` через параметр `--commit-hash`. mctl-web использует wrangler для деплоя в GitHub Actions (`deploy.yml`), что делает уязвимость прямым вектором атаки на CI/CD-пайплайн. Фикс тривиален: bump версии wrangler до v4.59.1 или выше.
**Source:** https://github.com/advisories/GHSA-36p8-mvp6-cv38

### 2. nuxt-upgrade-4-4-2: Обновление Nuxt до 4.4.2 и миграция на vue-router v5
**Категория:** refactor
**Impact:** 4
**Effort:** 3
**Обоснование:** vue-router v4.6.4 — финальный релиз ветки v4 (EOL); Nuxt 4.4.2 требует vue-router v5, и откладывание перехода накапливает технический долг по критической зависимости роутинга. Переход даёт также ускорение роутинга до 28x (unrouting) и типизированные layout props. Мажорное обновление vue-router требует проверки маршрутной конфигурации и composables, но объём кода у mctl-web небольшой (3 страницы).
**Source:** https://github.com/nuxt/nuxt/releases/latest; https://github.com/vuejs/router/releases/latest

### 3. vue-patch-3-5-33: Обновление Vue до 3.5.33
**Категория:** performance
**Impact:** 2
**Effort:** 1
**Обоснование:** Патч-релиз Vue 3.5.33 без breaking changes закрывает накопленные исправления с версии 3.5.30. Обновление безрисковое, занимает минуты и является обязательной гигиеной зависимостей перед более крупным обновлением Nuxt/vue-router.
**Source:** https://github.com/vuejs/core/releases/latest
